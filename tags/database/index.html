In this post, Lets build an awesome todo app for our legendary comic Heroes. What is Sequelize: Sequelize is a promise-based ORM for Node.js and io.js. It supports the dialects PostgreSQL, MySQL, MariaDB, SQLite and MSSQL and features solid transaction support, relations, read replication and more.
